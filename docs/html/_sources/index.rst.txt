セキュリティのアレまとめ
=========================================
.. rubric:: このサイトはなに？

* 日本初の情報セキュリティ系ポッドキャスト「セキュリティのアレ」に出てくる情報を個人的にまとめたサイトです。
* ポッドキャストを楽しむためのまとめであり、情報セキュリティの専門家ではないので、正確性・妥当性を保証するものではありません。あらかじめご了承ください。
* お問合せ・ご意見・ご要望は `Twitter@BerandaMegane <https://twitter.com/BerandaMegane>`_ へお願いします。
